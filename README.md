# Test TypeScript Project

This project is a TypeScript-based Node.js application with multiple tasks, each located in its own Git branch.

## 📦 Project Setup

- **TypeScript** — Static typing for JavaScript  
- **Node.js** — JavaScript runtime  
- **Nodemon** — Automatic restarts during development  
- **Rimraf** — Clean build directories  
- **ts-node** — Run TypeScript directly  
 


## 🏗 Project Structure

- `src/` — Your TypeScript source code  
- `build/` — Compiled JavaScript output after running `npm run build`

## 💾 Installation

```bash
git clone https://github.com/sentares/Result-Typescript-Test.git
cd test-typescript
````

install dependencies:

```bash
npm install
```

## 📁 Scripts

| Command         | Description                                           |
|-----------------|-------------------------------------------------------|
| `npm run dev`   | Run the project in development mode with `nodemon`    |
| `npm run build` | Compile TypeScript to JavaScript (output in `build/`) |
| `npm start`     | Build and run the project (`node build/index.js`)     |
